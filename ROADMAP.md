
# Roadmap

## Applications

1. **GitBucket**
     - Self-hosted Git repository service.

2. **Tennis Court Booking**
     - Booking and scheduling system for tennis courts.
     - Single club: stratford park tennis court

3. **Personal Data Governance Platform**
     - Centralized aggregation, backup, analysis, and visualization of personal data.
     - Data domains:
         - Finance
            - banking
            - pension
            - investments
            - belongings
         - Health data
             - Medical records
             - Fitness tracker data
             - Exercise advice
         - Contacts
             - phone
             - mail
             - email
             - social media
             - platforms like Gumtree
         - Time management
             - Calendar
         - Learning
         - Reading
         - Playlist
         - Watchlist
